# medi_talk_for_flutter

一个在线问诊的移动端应用，可以选择医生进行在线问诊<br>
当然这个只是我在工作之余学习flutter写的，有什么bug请大家多多包涵

## 使用

```
# 第一步先clone下来
git clone https://github.com/RogerPeng123/medi_talk_for_flutter.git

# 第二步执行加载必要的包
flutter pub get


# 然后就可以用在你的虚拟机或者手机上面了
```
